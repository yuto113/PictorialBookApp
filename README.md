# PictorialBookApp

このプロジェクトは、FlaskベースのウェブアプリをTkinterを使ったデスクトップアプリに変換したものです。

## 実行方法

1. Python環境を設定（仮想環境推奨）
2. 依存関係をインストール: `pip install -r requirements.txt`
3. DB初期化: `python init_db.py` （テストユーザー: name='test', password='test'）
4. デスクトップアプリを実行: `python main.py`

## 機能
- ログイン/ログアウト
- ユーザーページ: 日付データの表示、検索、フィルタ（公式/ユーザー）、いいね
- 日付詳細: 情報表示、チャット機能
- ポップなデザイン（ピンク背景、赤テキスト）

## 注意
- データベースはFlaskと同じものを使っています。
